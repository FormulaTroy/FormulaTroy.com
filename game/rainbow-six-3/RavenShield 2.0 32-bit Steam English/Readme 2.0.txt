-1- Introduction

Ravenshield 2.0 is the definitive community-designed patch for upgrading and also
improving Ravenshield's core gameplay and graphics.  Over years of dedication and
extensively hard work, we are happy to present the following amazing new features:

-Improved RAINBOW Skins.
-Improved Hostile Skins.
-Improved Hostage Skins.
-Improved First Person Weaponry Skins.
-Improved Third Person Weaponry Skins.
-Improved HOSTILE Artificial Intelligence.
-Improved RAINBOW Artificial Intelligence.
-Improved Special Effects during Gameplay.
-Improved damage system on shotgun munition.
-Improved Map Templates, with Realistic AI.
-Improved Magazine Count, for added Realism.
-Unique Ironsights for individual weaponry.
-Unique Scope sights for individual weaponry.
-Unique Goggle HUD, with active damage display.
-Removed the ability to sight in during NVG use.
-Removed 90% of Frag Grenade Use by Terrorist AI.
-Removed Crosshair's on all weapons, gadgets, etc.
-Redesigned Menu, with a cleaner design for Raven Shield.
-Redesigned Menu, with a cleaner design for both add-ons.
-New Map Loading textures, with a matching cleaner design.
-Added the ability to use all add-on weapons at all times.
-Much, Much more.

-2- Installation

STEP 01
LOCATE the folder your current RavenShield installation is in, usually within the
C:\Program Files\Red Storm Entertainment\ or C:\Steam\steamapps\common and find a
folder either named "RavenShield" for retail disc users, or "Rainbow Six 3 Gold".

STEP 02
COPY the folder "RavenShield" or "Rainbow Six 3 Gold" to a new location, preferably
in a new folder, this will become your backup folder, name it something appropriate.
This folder is important to keep, in case of an emergency or in the more unlikely of
events, dislikes Raven Shield 2.0 and wishes to copy the files back, after uninstall.

STEP 03
Run the installer provided, and ensure you specify the folder that your Raven Shield
is in, do NOT select RavenShield or Rainbow Six 3 Gold, merely specify the folder it
is installed in, in the case of regular disc it will be Red Storm Entertainment, the
steam version is within Steam\steamapps\common, select common. Usually the installer
will already have this specified, and you may automatically click next to run this.

STEP 04
CONGRATULATIONS! You should now have Raven Shield 2.0 installed into your file, if at
any point the installation did not complete, you may also elect to install this to an
alternative location and just simply copy the contents of these files over to RVS 1.6!
=============================ALTERNATIVE DUAL INSTALLATION=============================
STEP 01
LOCATE the folder your current RavenShield installation is in, usually within the
C:\Program Files\Red Storm Entertainment\ or C:\Steam\steamapps\common and find a
folder either named "RavenShield" for retail disc users, or "Rainbow Six 3 Gold".

STEP 02
COPY the folder "RavenShield" or "Rainbow Six 3 Gold" to a new location, preferably
in a new folder, this will become your second folder, name it something appropriate.
This folder is important to keep if you wish to have both a RavenShield 1.6 *AND* an
installation of RavenShield 2.0! Just ensure you name either one appropriately also.

STEP 03
Run the installer provided, and ensure you specify the folder you want a RavenShield
2.0 installation to be in, likely to be the new folder you created in STEP 02. Fully
installed onto the containing folder of the newly created RavenShield 2.0 folder will
ensure you have a clean version of RavenShield 1.6, and a clean version of 2.0 as well.
You may also create shortcuts from either one, and appropriately name these if needed!

STEP 04
CONGRATULATIONS! You should now have Raven Shield 2.0 installed into your file, if at
any point the installation did not complete, you may also elect to install this to an
alternative location and just simply copy the contents of these files over to folders!

NOTE: The installers are set up to install automatically into their "usual" location,
however if you have chosen a custom location, please ensure you install it correctly,
the installer *must* point to a location CONTAINING RavenShield, but do not install it
into RavenShield itself, as this will overwrite any of the previous 1.6 installations.
